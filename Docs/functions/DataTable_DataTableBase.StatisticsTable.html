  • Function to dynamically create and display a HTML table based on solver's time statistics.

    Remarks

    This function generates a new table displaying statistical metrics for different solvers. The table is added to the 'statisticsTable' HTML div. For each solver, it creates a row with the solver's name in the first column, and the solver's statistics in the subsequent columns. Note: this function directly manipulates the DOM and doesn't return anything.
