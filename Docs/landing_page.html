VIVA-APP

A free, open-source application designed for visualizing, interacting with, verifying, and analyzing benchmarking data for optimization solvers. It presents data in a tabular format and offers various plotting options to enhance understanding and interpretation of the data.

Download Web Version

Features

Interactive Table

Provides an overview of results from benchmark files, allowing dynamic interaction.

Versatile Charts

Includes eight different chart types for comprehensive data visualization.

Comparison Page

Allows easy comparison of different solvers and solution time per problem.

Save Functionality

Allows saving filtered results to local storage for quick access.

Advanced Filtering

Features a powerful filtering system with custom search queries and per-column filtering options.

Performance

Performant when visualizing large benchmarking result sets.

Problem Libraries

In-built popular problem libraries for relevant information integration.

Open Source

The source code is open for modification and inspection.
